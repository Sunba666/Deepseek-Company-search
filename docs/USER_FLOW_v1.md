## 四、页面跳转逻辑

### 4.1 全局导航

```
[Logo]  [公司]  [岗位]  [内推]  [AI 推荐]  [投递管理]  [AI Copilot]  [登录/用户]
                                                          |
                                                     [AI 搜索框]
```

### 4.2 核心最短路径

```
路径 A (3次点击): 首页 -> 岗位搜索结果 -> 岗位详情 -> 复制内推码 -> 官网跳转
路径 B (5次点击): 首页 -> 公司列表 -> 公司详情 -> 岗位列表 -> 岗位详情 -> 投递
路径 C (5次点击): 首页 -> AI 推荐 -> 上传简历 -> 匹配结果 -> 岗位详情 -> 投递
路径 D (对话式):   AI Copilot -> 推荐上海AI产品经理 -> 结果 -> 详情 -> 投递
```

### 4.3 用户状态流转

```
未登录用户 -> 浏览有限内容
    +-- 点击收藏/投递 -> 登录/注册
            +-- 新用户：选择岗位分类 -> 个性化首页
            +-- 老用户：首页（个性化推荐 + 继续进度）
```

### 4.4 弹窗/侧边栏

| 触发 | 内容 | 关闭 |
|------|------|------|
| 点击 AI 分析 | JD 分析弹窗 | ESC / 外部点击 |
| 复制内推码 | Toast 已复制 | 自动消失 |
| 点击投递 | 确认弹窗 | 确认/取消 |
| 点击 AI Copilot | 侧边栏展开 | 关闭按钮 |

## 五、用户操作流程

### 5.1 新用户首次访问

```
进入首页
  +-- 浏览热门公司/岗位（无需登录）
  +-- 点击岗位 -> 查看详情
  +-- 点击收藏/投递 -> 弹窗引导登录
  +-- 注册 -> 选择关注分类 -> 个性化首页
```

### 5.2 岗位筛选

```
进入 /jobs
  +-- 分类筛选: 研发/AI/产品/运营/设计...
  +-- 子分类: Java/Golang/Python/前端/React...
  +-- 城市/经验/学历/薪资/公司规模/融资阶段
  +-- 关键词搜索（支持自然语言）
  +-- 结果列表 -> 排序/视图切换 -> 点击进入详情
```

### 5.3 AI 推荐

```
AI 推荐页
  +-- 上传简历（PDF/Word）-> AI 解析 -> 匹配岗位列表
  +-- 手动输入技能/经验 -> AI 匹配
      +-- 结果: 岗位名/公司/匹配度/预计薪资/推荐理由 -> 点击投递
```

### 5.4 投递管理 CRM

```
看板视图（Kanban）
  [收藏] -> [准备投递] -> [已投递] -> [HR 查看]
  -> [笔试] -> [一面] -> [二面] -> [HR 面] -> [Offer] / [拒绝]
  拖拽切换状态 + 统计看板（投递数/回复率/面试率/Offer率）
```

### 5.5 AI Copilot

```
浮动按钮 -> 侧边栏展开
  +-- 推荐上海AI产品经理 -> 岗位列表
  +-- 我想进字节跳动 -> 推荐部门/内推人/简历建议
  +-- 分析这个JD -> 当前页JD分析
  +-- 优化简历 -> 跳转简历优化
```

A. 公司信息: 基本信息 | 招聘信息 | 薪资数据 | 内推 | 面经 | AI 画像
